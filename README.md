# Finventory Vercel Website

This is a Vite + React + Tailwind website for Finventory.app.

## Local setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Create a GitHub repo.
2. Upload all files in this folder.
3. In Vercel, click "Add New Project".
4. Import the GitHub repo.
5. Use the default Vite settings.
6. Deploy.
7. Add your custom domain: Finventory.app.

## Notes

The beta signup form is currently a visual placeholder. Connect it later with Formspree, ConvertKit, Supabase, Firebase, or a Vercel serverless function.
